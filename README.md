# Nintendo Zone HTML Template
Edit the included files and create your own Nintendo Zone content. You can also create new HTML files.

Because of the file size, the `.moflex` files are not included. You can find them in the SatellaZone download.

## HELP WANTED
I have this great idea for a computer program that lets you easily host your own custom Nintendo Zone hotspot without needing a console to use for distribution. It would definitely work on Linux (with low-level access to Wi-Fi hardware), and it would be great if it could work on Windows as well (maybe by using the same method that Dolphin uses to connect to Wii Remotes, or some other way to emulate Nintendo's local wireless protocol).
However, I don't know how to make this program. I don't even know where to start. That's partially why I created this repository: so that other people will have some content to use with a custom Nintendo Zone if someone makes a program like that.

If you know how to (or you can help develop it or get it started), please make a program like this. You do not need to give me credit for the idea, but I would like it if you told me about your program (with a link to the GitHub repo) if you get to the point where it's mostly working and has a functional release (just so I'm aware of its existance and can test and use it). I thought of the name "SpotZone" (a mix of SpotPass and Nintendo Zone), but feel free to call your program whatever you want.

If you know anyone that could make this program/project, please tell them about my idea.

If someone ends up starting this project, I would be happy to help with it. I could help decide what it would look like and how you would use it, or I could create assets for it (such as a logo/icon or a printable poster for people to put up at their Nintendo Zone location).

### More Direct Help
